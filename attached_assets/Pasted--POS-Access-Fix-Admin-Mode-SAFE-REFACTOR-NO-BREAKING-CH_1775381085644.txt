## 2️⃣ إضافة وضع "اختيار الفرع للمدير"

---

### 🧠 السلوك المطلوب:

إذا:

* المستخدم = admin
* ولا يوجد warehouse أو safe

👉 يظهر له شاشة اختيار:

---

### 🎨 UI جديد:

#### Component: `AdminPOSSetup`

يحتوي على:

* Dropdown اختيار الفرع (warehouse)
* Dropdown اختيار الخزينة (safe)
* زر: "بدء نقطة البيع"

---

### 🧱 الكود:

```tsx
function AdminPOSSetup({ onStart }: { onStart: (w: number, s: number) => void }) {
  const { data: warehouses = [] } = useQuery(...)
  const { data: safes = [] } = useGetSettingsSafes()

  const [warehouseId, setWarehouseId] = useState<number | null>(null)
  const [safeId, setSafeId] = useState<number | null>(null)

  return (
    <div className="fixed inset-0 flex items-center justify-center bg-black/90" dir="rtl">
      <div className="bg-[#0f172a] p-6 rounded-2xl w-full max-w-md space-y-4">

        <h2 className="text-white text-xl font-bold text-center">
          اختيار الفرع والخزينة
        </h2>

        <select onChange={e => setWarehouseId(Number(e.target.value))}>
          <option>اختر الفرع</option>
          {warehouses.map(w => (
            <option key={w.id} value={w.id}>{w.name}</option>
          ))}
        </select>

        <select onChange={e => setSafeId(Number(e.target.value))}>
          <option>اختر الخزينة</option>
          {safes.map(s => (
            <option key={s.id} value={s.id}>{s.name}</option>
          ))}
        </select>

        <button
          disabled={!warehouseId || !safeId}
          onClick={() => onStart(warehouseId!, safeId!)}
          className="w-full bg-amber-500 py-3 rounded-xl font-bold"
        >
          بدء البيع
        </button>

      </div>
    </div>
  )
}
```

---

### ✨ تعديل الـ render:

```tsx
if (isAdmin && (!warehouseId || !safeId) && !adminSetup.warehouseId) {
  return (
    <AdminPOSSetup
      onStart={(w, s) => setAdminSetup({ warehouseId: w, safeId: s })}

  )
}
```
