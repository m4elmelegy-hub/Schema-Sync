أنشئ تطبيق ويب متكامل باستخدام Node.js + Express يعمل على Replit لإدارة كشف حساب العملاء، وفق المواصفات التالية:

[التقنيات]
- Backend: Node.js + Express
- Database: SQLite أو MongoDB
- PDF Generator: Puppeteer
- WhatsApp API: Twilio (اختياري)
- Frontend: HTML + CSS + Bootstrap

----------------------------------------

[قاعدة البيانات]

أنشئ الجداول التالية:

1. customers
- id
- name
- phone
- created_at

2. invoices
- id
- customer_id
- total_amount
- paid_amount
- remaining_amount
- status (paid / unpaid / partial)
- created_at

3. payments
- id
- customer_id
- amount
- method (cash / transfer / card)
- note
- created_at

----------------------------------------

[المنطق المطلوب]

1. عند إضافة فاتورة:
- يتم ربطها بالعميل
- تحديث الرصيد

2. عند تسجيل دفعة:
- يتم خصمها من الفواتير تلقائيًا
- تحديث حالة الفواتير:
  - مدفوع
  - جزئي
  - غير مدفوع

3. إنشاء كشف حساب تلقائي:
- عند طلب كشف حساب لعميل:
  - يتم جلب جميع الفواتير + الدفعات
  - حساب:
    - إجمالي المبيعات
    - إجمالي المدفوع
    - الرصيد المتبقي

----------------------------------------

[PDF احترافي]

أنشئ PDF باستخدام Puppeteer بتصميم احترافي:

- اتجاه RTL (عربي)
- هيدر:
  - شعار الشركة
  - عنوان: كشف حساب عميل

- صندوق بيانات العميل

- كروت ملخص:
  - إجمالي المبيعات (أخضر)
  - المدفوع (أزرق)
  - المتبقي (أحمر)

- جدول الفواتير:
  - ألوان للحالة:
    - أخضر = مدفوع
    - أحمر = غير مدفوع
    - أصفر = جزئي

- جدول الدفعات

- Footer:
  "هذا كشف حساب صادر آليًا"

----------------------------------------

[API Endpoints]

- POST /customers
- POST /invoices
- POST /payments
- GET /statement/:customerId (يرجع PDF)

----------------------------------------
