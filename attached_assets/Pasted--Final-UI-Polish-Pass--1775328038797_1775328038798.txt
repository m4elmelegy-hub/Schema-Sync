المرحلة الحالية ممتازة، الآن أحتاج "Final UI Polish Pass" على النظام بالكامل.

ركز فقط على اللمسات النهائية لتحويله من جيد جدًا إلى SaaS-level احترافي.

========================

1. Spacing Consistency
   ========================

* تأكد أن كل الصفحات تستخدم نفس نظام المسافات
* اعتمد scale ثابت (8 / 12 / 16 / 24 / 32)
* لا يوجد اختلاف عشوائي بين الصفحات

========================
2) Card System توحيد كامل
=========================

* كل الكروت تستخدم نفس:

  * border radius
  * padding
  * shadow
* لا يوجد أكثر من style للكروت
* استخدم erp-card فقط

========================
3) Topbar Polish
================

* تحسين التباين بين الخلفية والمحتوى
* جعل search أوضح بصريًا
* تحسين توزيع العناصر (spacing + alignment)
* توحيد ارتفاع كل عناصر الـ topbar

========================
4) Sidebar Polish
=================

* توحيد ارتفاع كل menu items
* تحسين المسافة بين sections
* جعل active item أوضح لكن بدون مبالغة
* تقليل العناصر البصرية غير الضرورية

========================
5) Interaction Consistency
==========================

* كل hover نفس السلوك (translateY بسيط + background)
* كل clickable element له نفس feedback
* إزالة أي animation غير متناسق

========================
6) Typography Hierarchy
=======================

* توحيد أحجام:

  * titles
  * subtitles
  * body text
* تحسين line-height للقراءة

========================
7) Visual Hierarchy
===================

* أهم عنصر في الصفحة واضح فورًا
* تقليل التنافس بين العناصر
* إبراز primary actions

========================
8) Remove Visual Noise
======================

* تقليل borders الزائدة
* استخدام spacing بدل lines
* تبسيط الواجهة

========================
9) Micro Improvements
=====================

* تحسين focus states
* تحسين loading states (لو موجودة)
* تحسين empty states

========================
الهدف النهائي:
==============

واجهة:

* هادئة بصريًا
* واضحة جدًا
* متناسقة 100%
* تشبه SaaS عالمي (Stripe / Notion style)

بدون إضافة features جديدة
فقط polish وتحسين التفاصيل
