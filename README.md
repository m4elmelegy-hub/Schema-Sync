# Schema-Sync ERP System

A comprehensive, production-ready **multi-tenant ERP system** for modern businesses.

## Features
- Sales Management, Purchase Management, Inventory
- Double-entry Accounting with General Ledger
- JWT Authentication + 2FA
- Role-Based Access Control (5 roles, 30+ permissions)
- WCAG AA Accessibility
- 90+ Tests with 75%+ Coverage
- Production Ready

## Quick Start
\\\ash
git clone https://github.com/m4elmelegy-hub/Schema-Sync.git

pnpm --filter @workspace/erp-system dev
